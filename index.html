mac上面 git 搭配 hexo完成个人博客搭建.

想开始自己记录点自己学习的点滴进程,觉得程序员这个东西还是得自己写一写技术博客这个东西,就当云笔记来用吧.那第一篇文章要写的话就从怎么搭建的blog开始写吧.
 mac 上面如何搭建一个hexo环境呢?
首先我们比较穷,我们用git来当我们的服务器.

  1. 全是静态文件，访问速度快；
  2. 免费方便，不用花一分钱就可以搭建一个自由的个人博客，不需要服务器不需要后台；
    3.可以随意绑定自己的域名，不仔细看的话根本看不出来你的网站是基于github的；
  3. 数据绝对安全，基于github的版本管理，想恢复到哪个历史版本都行；
  4. 博客内容可以轻松打包、转移、发布到其它平台;

####下载一个git

那么从最开始,我们需要一个git 账户,申请一个.
新建一个仓库（new repository），命名一定是你的github的用户名(username)加上.github.io，这个仓库的全称命名就是username.github.io，然后点击create repository
然后去下载一个git
下完了安装,nextStep点下去

####下载Node.js
Node.js 下载地址
选一个即可，下载后安装时一路默认next(安装路径可以自选)。

####开始部署hexo
打开终端
接着 sudo npm install hexo-cli -g
输入你的开机密码,等待读条完成后
再次输入sudo npm install hexo –save
等待安装完成。

找个地方新建一个文件夹命名为”hexo”,我是在user目录下.
打开终端 cd 到该文件夹路径下
输入hexo init，请等待，这个时间可能长可能短，等到下一条可输入命令出现的时候才算安装好了，中途如果强行退出请在该文件夹重新hexo init

####检测你的git node npm hexo是否安装和他们的版本信息

打开终端 依次输入git –version,node -v,npm -v,hexo version，依次检测 Git node npm 以及 hexo 的版本。

####在_config.yml,进行基础配置
修改一些你的博客名字、描述、作者等，修改后的效果大家可以看看。同时大家可以修改自己的主题，更多主题点击这里。修改主题就修改_config.yml里面的theme.
改完之后打开终端 cd 到该路径下
输入hexo g
然后输入hexo s，启动本地服务器，好了，你可以在你电脑的服务器看到你的博客了。请打开你的浏览器，输入http://localhost:4000/，你就看到你的博客样子啦。停止你的本地服务器，你可以在终端命令页面按住Ctrl+C来停止.

####hexo与github关联
还是打开你的_config.yml文档，将最下面的内容修改一下，type改成git，repo后面加上你的仓库的ssh.

配置完成后保存，终端 cd到该hexo目录下,
输入npm instal hexo-deployer-git –save
将你的代码推到网上去：依次出入hexo clean,hexo g,hexo d
.稍等5-15分钟，然后打开浏览器输入你的username.github.io，你就可以看到你的博客啦

###发表文章
你要写文章，就在hexo\source_posts文件夹里面创建.md文档就好啦(关于md可以百度教程简单学学，基本格式十分钟就学会啦)。写完了保存后，依旧是hexo g,hexo d，就上传到你的博客了。
